--Name add--
Adds users to list.
/na [user] * n
/na tvrtko karlo jura

--Name remove--
Removes user, only if balance = 0.
/nr [user]
/nr tvrtko

--Name change--
Changes username
/nc [old username] [new username]
/nc tvrtko tvrtkonator

--Group create--
Creates group
/gc [user] * n
/gc party karlo tvrtko jura grgur

--Group remove--
Removes group
/gr [group name]
/gr party

--Group print--
Returns a list of groups
/gp

--Transaction--
Make transaction
/t [surplus user] ([deficit user] [amount]) * n
/t tvrtko karlo 10
/t tvrtko karlo 10 grgur 20 jura 15

--Transaction division--
Make transaction that divides deficit among all people listed (including surplus user).
/td [surplus user] [deficit user] * n [amount]
/td tvrtko karlo grgur jura 1000

--Transaction group--
Functions like Transaction division but with preselected users saved in a group.
/tg [surplus user] [group name] [amount]
/tg tvrtko party 500

--State force--
Allows user to set the state manually. Inbalance will be divided among users.
/sf ([user] [balance]) * n
/sf tvrtko 10 grgur 5 jura -5 karlo -10

--State--
Returns state
/s

--Undo--
Undoes last transaction (Only transactions).
/u

--Random--
Returns random user from a list
/r

--Custom command--
Returns output for custom command
/[custom command]

--Custom commands list--
Returns list of custom commands
/cl

--Command create--
Creates custom command
/cc [command] [output]

--Command remove--
Removes custom command
/cr [command]

--Help--
Returns help message
/h

--Command tips 'n' tricks--
You can put math expresions as amount. 
/t tvrtko karlo 5*7.53

You can put '#' to comment your transaction.
/t tvrtko karlo 20 #money for lunch

Spaces, tabs and line breaks do not matter.
/t tvtko   karlo 
19       +17
#lunch
